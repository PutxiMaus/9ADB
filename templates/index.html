Bootloader / Fastboot

                  
Modo bootloader para recuperar, consultar y flashear imágenes. Usa USB siempre que puedas.

                

                
              

              

                
                
                
                
              

              
Pulsa Actualizar fastboot.

            


            

              
Información

              

                
                
                
              

              

                
                
              

            


            

              
Slots A/B

              
Cambiar slot puede impedir arrancar si el otro slot no está bien. Requiere confirmación.

              

                
                
              

            


            

              
Boot temporal

              
Ejecuta una imagen sin flashearla permanentemente: fastboot boot imagen.img.

              
              
            


            

              
Flash imagen

              
Zona delicada. Elige partición e imagen. Para ejecutar debes escribir FLASH.

              

                
                
                
              

              
            


            

              
Comando manual

              
Escribe argumentos sin la palabra fastboot. Ejemplo: getvar product.

              

                
                
                
              

            


            

              Zona peligrosa: unlock / lock / erase
              

                
Bootloader lock / unlock

                
Puede borrar datos o dejar el móvil sin arrancar si no sabes lo que haces. Confirmación requerida: BORRAR DATOS.

                

                  
                  
                  
                  
                

              

              

                
Erase partición

                
Borra una partición. Confirmación requerida: ERASE.
